# Free Drag Crop

A professional, high-precision interactive cropping node for ComfyUI.

## Description
This node allows you to perform pixel-perfect cropping directly on the ComfyUI canvas. It supports:
- 8-point interactive resizing
- Aspect ratio locking
- Smart presets (9:16, 16:9, etc.)
- Bilingual help system

## Features
- **Real-Pixel Precision**: Operates on actual image dimensions.
- **Master Resets**: Quickly expand or clear selections.
- **Adaptive UI**: High-fidelity sidebar for instructions.

## Tutorial
1. Connect an image to the node.
2. Drag the handles on the canvas to define your crop area.
3. Use the sidebar buttons to lock ratios or reset to full image.
4. The output will be the cropped portion of the image.
